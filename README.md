# Sistema de Gestión y Control de Inventario Tecnológico

Este es el prototipo funcional de la interfaz para el sistema de control de activos de TI y Telecomunicaciones.

## Características Implementadas
- **Dashboard Corporativo**: Métricas críticas del parque tecnológico y alertas de mantenimiento preventivo.
- **Gestión de Inventario (Data Grid)**: Visualización detallada de equipos con filtrado y búsqueda.
- **Asignación de Responsables**: Vista interactiva para vincular equipos a empleados y áreas físicas.
- **Historial de Mantenimiento (Timeline)**: Trazabilidad completa de intervenciones preventivas y correctivas.
- **Generación de Reportes**: Interfaz de exportación con filtrado por categoría y fechas.

## Stack Tecnológico
- **Frontend**: React 19 + Vite 8.
- **Estilos**: Vanilla CSS con variables modernas para diseño premium.
- **Iconos**: Lucide React.
- **Animaciones**: Framer Motion para micro-interacciones suaves.

## Instrucciones de Instalación
1. Clonar o descargar este repositorio.
2. Ejecutar `npm install` para instalar las dependencias.
3. Ejecutar `npm run dev` para iniciar el servidor de desarrollo.
